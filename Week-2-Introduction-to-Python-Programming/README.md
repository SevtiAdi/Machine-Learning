# Week 2 – Pengenalan Pemrograman Python

## Deskripsi

Pada minggu ini kita mempelajari dasar-dasar pemrograman menggunakan bahasa **Python**. Python merupakan salah satu bahasa pemrograman yang sangat populer dan banyak digunakan dalam berbagai bidang seperti **data science, machine learning, artificial intelligence, robotics, dan pengembangan software**.

Materi ini bertujuan untuk memperkenalkan konsep dasar Python serta cara penulisan sintaks yang benar sehingga dapat digunakan sebagai dasar untuk mempelajari topik yang lebih lanjut.

## Tujuan Pembelajaran

Setelah mempelajari materi ini, diharapkan mahasiswa dapat:

* Memahami dasar-dasar sintaks Python
* Memahami penggunaan variabel dan tipe data
* Memahami konsep fungsi dalam Python
* Memahami prinsip dasar pemrograman Python
* Memahami konsep dasar **Object-Oriented Programming (OOP)**

## Isi Folder

Folder ini berisi beberapa materi praktikum dan latihan, yaitu:

* **Hands_On_Function_in_Python**
  Berisi latihan penggunaan fungsi dalam Python seperti pembuatan fungsi dan penggunaan parameter.

* **Hands_On_Main_Principles_of_Python**
  Menjelaskan prinsip-prinsip dasar dalam pemrograman Python.

* **Hands_On_OOP_Paradigm**
  Berisi latihan mengenai konsep **Object-Oriented Programming (OOP)** seperti class dan object.

* **Week2_Tipe_Data.ipynb**
  Notebook yang menjelaskan berbagai **tipe data dalam Python** seperti integer, float, string, list, dan dictionary.

## Tools yang Digunakan

Beberapa tools yang digunakan dalam pembelajaran ini antara lain:

* Python 3

* GitHub

## Konsep Python yang Dipelajari

Beberapa konsep dasar Python yang diperkenalkan dalam minggu ini antara lain:

* Variabel

## Kesimpulan

Pemahaman dasar Python sangat penting sebelum mempelajari topik yang lebih lanjut seperti **data analysis, machine learning, artificial intelligence, dan robotics programming**. Materi dan latihan pada minggu ini memberikan fondasi dasar untuk memahami pemrograman Python secara lebih mendalam.
