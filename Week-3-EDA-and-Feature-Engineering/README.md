# Week 3 – Exploratory Data Analysis (EDA) dan Feature Engineering

## Deskripsi

Pada minggu ini kita mempelajari **Exploratory Data Analysis (EDA)** dan **Feature Engineering** yang merupakan tahap penting dalam proses pengolahan data sebelum membangun model machine learning.

Exploratory Data Analysis digunakan untuk memahami karakteristik data melalui proses analisis statistik dan visualisasi data. Sedangkan Feature Engineering digunakan untuk mempersiapkan dan meningkatkan kualitas fitur agar model machine learning dapat bekerja dengan lebih baik.

## Tujuan Pembelajaran

Setelah mempelajari materi ini, diharapkan mahasiswa dapat:

* Memahami konsep **Exploratory Data Analysis (EDA)**
* Menganalisis struktur dan distribusi data
* Mengidentifikasi missing values pada dataset
* Melakukan pembersihan data (data cleaning)
* Memahami konsep dasar **Feature Engineering**
* Melakukan transformasi data untuk meningkatkan kualitas fitur

## Materi yang Dipelajari

Beberapa topik yang dipelajari dalam minggu ini antara lain:

* Pengenalan **Exploratory Data Analysis (EDA)**
* Analisis statistik dasar pada dataset
* Visualisasi data
* Penanganan missing values
* Encoding data kategorikal
* Transformasi dan pembuatan fitur baru

## Tools yang Digunakan

Tools yang digunakan dalam materi ini antara lain:

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib / Seaborn

## Manfaat EDA dalam Machine Learning

EDA memiliki peran penting dalam proses machine learning, diantaranya:

* Memahami pola dan distribusi data
* Mengidentifikasi outlier
* Menemukan hubungan antar variabel
* Membantu menentukan metode preprocessing yang tepat

## Kesimpulan

Exploratory Data Analysis dan Feature Engineering merupakan langkah penting sebelum membangun model machine learning. Dengan memahami struktur dan kualitas data terlebih dahulu, model yang dihasilkan dapat menjadi lebih akurat dan lebih optimal.
