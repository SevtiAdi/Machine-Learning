# 🤖 Machine Learning

## 📌 Overview

Machine Learning adalah cabang dari kecerdasan buatan (Artificial Intelligence) yang memungkinkan komputer untuk belajar dari data tanpa diprogram secara eksplisit. Dengan menggunakan algoritma tertentu, sistem dapat mengenali pola, membuat prediksi, dan mengambil keputusan secara otomatis.

Repository ini berisi kumpulan pembelajaran, eksperimen, dan implementasi konsep dasar hingga lanjutan dalam machine learning.

---

## 🎯 Tujuan

* Memahami konsep dasar machine learning
* Mempelajari berbagai algoritma machine learning
* Mengimplementasikan model dari data nyata
* Melakukan evaluasi dan analisis model

---

## 🧠 Jenis-Jenis Machine Learning

### 1. Supervised Learning

Model dilatih menggunakan data yang sudah memiliki label.

Contoh:

* Regresi (prediksi nilai kontinu)
* Klasifikasi (prediksi kategori)

---

### 2. Unsupervised Learning

Model belajar dari data tanpa label untuk menemukan pola tersembunyi.

Contoh:

* Clustering
* Dimensionality Reduction

---

### 3. Reinforcement Learning

Model belajar melalui interaksi dengan lingkungan menggunakan sistem reward dan punishment.
